# permutations

vector of permutations
# To download:
```sh
git clone https://github.com/bashmachnikov/permutations
cd permutations
```
# To build:
```sh
mkdir build
cd build
cmake ..
make
```

#to Run:
```sh
./permutations 1
```

Example:
```
[~/src/permutations/build]$ ./permutations 2
result
0: 2,1,0,
1: 1,0,2,
2: 1,2,0,
3: 2,0,1,
4: 0,1,2,
5: 0,2,1,
```

! No input checks are done !
